# 🏍️ Lost Limb Riders — Official Website README

Welcome to the **Lost Limb Riders (LLR)** official website repository.  
This repo contains the source code for the LLR site—built for riders, supporters, and anyone who believes in riding with purpose.

---

## 💪 About Lost Limb Riders

**Lost Limb Riders** is a community that empowers amputees and differently-abled riders through:

- 📰 **Community updates & news**  
- 🏍️ **Rider stories and spotlights**  
- 🤝 **Support resources and links**  
- 📅 **Event information and meetups**  
- 🎗️ **Advocacy, awareness, and outreach**

Our mission: **Ride With Purpose.**

---

## 🎯 Website Goals

The LLR website is designed to:

- ⚡ Be fast, responsive, and accessible  
- 🎨 Reflect a bold, biker-inspired visual style  
- 📱 Work smoothly on mobile, tablet, and desktop  
- 🔧 Stay easy to maintain and expand as the community grows  

---

## 🧱 Tech Stack

This project uses:

- **HTML5**
